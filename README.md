# ORU
Origami pattern generation software
ORU Origami Studio is a single-file, offline-first browser tool for designing folded structures parametrically. It carries 18 fold families — origami tessellations (Miura, Yoshimura, Kresling, waterbomb, Resch, hypar and more), Engel folded-plate roofs and frame/arch systems, and deployable shelters (curved Miura vault, waterbomb magic-ball tube, radial gore dome, pleated tent). Every pattern animates from flat sheet to folded form, can be reshaped with a deform stack (twist, bend, taper, spherify), and exports to STL, OBJ, SVG and a print-ready flat crease pattern with mountain / valley / cut lines on real paper sizes. A built-in science panel gives the geometry and equations behind each form. Runs entirely in the browser, no install or build step.
